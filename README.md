# Project-IoMT

This repository contains code and machine learning models for analyzing IoMT (Internet of Medical Things) datasets for viral fever detection and prediction.

## Machine Learning Models Used

### Classification Models
1. Logistic Regression (LR)
2. Support Vector Machine (SVM)
3. K-Nearest Neighbor (KNN)
4. Decision Tree
5. Random Forest
6. Ensemble Method

## Methodology

The following methods have been implemented in this project:

1. **Dataset Collection** - Gathering IoMT data for viral fever cases
2. **Dataset Cleaning** - Data preprocessing and handling missing values
3. **Model Training** - Training and evaluating machine learning models

## Installation

To get started with this project, clone the repository and install the required dependencies:

```bash
git clone https://github.com/SaymaSJ/Porject-IoMT-.git
cd Porject-IoMT-
pip install -r requirements.txt
```

## Usage

1. Prepare your dataset in the appropriate format
2. Run the data cleaning scripts
3. Train the models using the provided notebooks/scripts
4. Evaluate model performance and predictions

## Results

The models are evaluated based on accuracy, precision, recall, and F1-score metrics.
